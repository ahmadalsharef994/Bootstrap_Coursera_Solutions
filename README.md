## 📓 Course Assignments

| Week | Topic |
|------|-------|
| Week 1 | Bootstrap grid, responsive layout, typography |
| Week 2 | Navigation, buttons, forms, tables |
| Week 3 | JavaScript components: modals, carousel, scrollspy |
| Week 4 | Bootstrap customization with Sass, build tools |
